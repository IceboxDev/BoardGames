# Sushi Go! Card Art

12 card illustrations (one per card type) used across the 108-card deck. Each card type shares a single illustration. The style matches the original Sushi Go! game by Gamewright — cute, kawaii-inspired cartoon food characters with personality, bold outlines, and cheerful saturated colors. Not realistic food photography.

The UI composites the card border, type label, scoring text, and maki/point indicators on top of these illustrations — the prompts generate **only the character art**.

---

## Shared Card Art Style Directive

> Kawaii-style cartoon illustration of a Japanese food character with a cute anthropomorphic face (small dot eyes, tiny smile, rosy cheeks). Bold clean outlines, flat color fills with soft cel-shading. The character is centered on a solid color background with a subtle radial gradient (lighter in the center, slightly darker at edges). The mood is cheerful and playful, like a children's card game. Inspired by the original Sushi Go! art style by Nan Rangsima — rounded, friendly shapes with exaggerated proportions. No text, no numbers, no card borders, no UI elements. Portrait 2:3 aspect ratio, 512×768 minimum.
