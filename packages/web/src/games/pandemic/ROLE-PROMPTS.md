# Pandemic Role Card Portraits

Portraits for the 7 Pandemic player roles, used in-game to identify the active player and display abilities. Each image is a bust portrait (chest-up) of the character in a dynamic professional pose.

The border color, role name header, pawn icon, and ability text are composited by the UI — these prompts generate **only the portrait illustration**.

Reference: original Pandemic (2013 2nd edition) role cards photographed for style matching.

---

## Shared Role Portrait Style Directive

> Semi-realistic digital portrait illustration, painterly with visible brushwork and soft color gradients. Chest-up bust composition, slight three-quarter angle. The character is in a dynamic pose — actively using equipment, gesturing, or scanning their surroundings. Dramatic side lighting with a warm rim light on one edge and soft fill on the other. The background is a monochromatic color wash matching the role's accent color, with subtle thematic silhouettes (city skylines, biohazard symbols, lab equipment) fading into the wash. Shallow depth of field blurs the background. No text, no logos, no UI elements, no card borders. Square 1:1 aspect ratio, 512×512 minimum.
